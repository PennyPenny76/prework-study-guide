Prework Study Guide

HTML

  • HTML is the structure of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is describing the styles and colors of all the elements.

git

  • git status:checks what branch we currently on
  • git checkout -b branch-name:creates a new branch and switches to it
  • git add-A:add all the changes
  • git commit -m"message":commit the changes and leave a messgae
  • git pull origin: pull the origin main branch back to the local enviroment.
  • git push origin -feature/name:push the changes to the feature branch.

Javascript

  • Javascript defines all the interactions between the elements on the webpage and the actions that the users make.
